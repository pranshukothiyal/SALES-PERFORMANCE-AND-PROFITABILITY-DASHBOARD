# SALES-PERFORMANCE-AND-PROFITABILITY-DASHBOARD

## PROJECT OBJECTIVE 
Developed an interactive Power BI dashboard to evaluate brand performance and profitability, delivering data-driven insights on sales patterns, regional growth, and product profitability. The dashboard empowers stakeholders to monitor key KPIs, uncover improvement opportunities, and drive strategic decisions for enhanced business outcomes.

## TOOLS AND TECHNOLOGIES 
Azure SQL Database- For data sourcing.
DAX - For advanced measures and calculations.
Power BI Desktop - For Data Visualization and dashboard creation.

## DASHBOARD FEATURES
**Interactive Filters & Slicers:**  
  Filter by region, product category, customer segment, or time range  
**Visuals Used:**  
  - Bar and Line Charts for sales trends  
  - Donut Charts for product mix analysis  
  - KPI Cards for quick metrics  
  - Maps for regional sales visualization

## HOW TO USE
1. Download the `.pbix` file from this repository.  
2. Open it in **Microsoft Power BI Desktop** (latest version recommended).  
3. Update data source credentials if required (Azure SQL).  
4. Refresh the dashboard to view updated insights
